# 工程下的GTest测试

根据该工程下gtest的部署，进行LLT测试的话，主要由以下四个步骤组成。

1. 切换到build目录下

`cd build`

2. 创建cmake系统

`cmake ../ .`

3. 编译

`cmake --build .`

4. 执行llt测试用例

`./tst/ExampleProject_tst`

这里的关键在于CMakelist.txt文件的内容及组织。该工程下无法通过ctest直接调用测试，这个后续再看如何改进，anyway，当前能够已经能够通过嵌入googletest成功调用gtest了。
